Comandos Git

    Começando o trabalho atualizando

  • Com finalidade de receber qualquer atualização anteriormente realizada
  • git pull

    • Criando uma nova feature:

  • git checkout develop
  • git checkout -b name-feature

    • Finalizar e fechar uma feature:

  • git checkout develop
  • git merge name-feature
  • git add .
  • git commit -m "texto"

    • Mandar para develop

  • git push

    • Apagar feature

  • git branch -d name-feature
